# 米游姬自定义光标套装

一套为《原神》和米游社粉丝制作的、基于“米游姬”形象的可爱Windows鼠标指针主题。

## 效果预览
观看完整安装与效果演示视频：[点击跳转至B站观看](https://www.bilibili.com/video/BV1hTLt6KE3k/?vd_source=1de2e30996b0006bfe2c3ad7bcfac367)

## 包含内容
- 普通选择指针
- 文本选择指针
- 忙状态指针
- 手型链接指针
- ...等全套常用指针样式

## 安装方法（Windows系统）
1. 下载本页面的 [`米游姬.zip`](./米游姬.zip) 文件并解压。
2. 打开解压后的文件夹，右键点击 `右键该文件，点击安装即可(ANSI)-常规.inf` 文件。
3. 在弹出的菜单中选择“安装”。
4. 在弹出的“鼠标属性”窗口中，切换至“指针”选项卡。
5. 在方案下拉菜单中，选择“**米游姬光标**”，点击“应用”即可。
6. 可以调整鼠标的光标大小，使得效果更佳

## 注意事项
- 本光标包仅适用于 **Windows** 操作系统。
- 安装时需要管理员权限，请放心，此操作是系统更换光标方案的标准流程。
